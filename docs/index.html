sync-cloud-storage

Synchronize files and directories between a remote computer and multiple Serverless cloud providers' storage.

Supported cloud providers:

  • [x] AWS S3

Features

  • Uses the latest official cloud provider's SDK
    • AWS S3: aws-sdk@3.x
  • Sync multiple storages at once by just defining patterns of glob strings to include or exclude files and directories
  • Set the following for each synced file:
    • prefix
    • access control list (ACL)
    • tags
    • metadata
  • Select a list of specific sync actions for each storage:
    • upload
    • delete

Installation

  1. ni: ni sync-cloud-storage -D
  2. npm: npm i sync-cloud-storage -D
  3. yarn: yarn add sync-cloud-storage -D
  4. pnpm: pnpm add sync-cloud-storage -D

Usage

Serverless

AWS S3

plugins:
  - sync-cloud-storage

custom:
  syncCloudStorage:
    - name: my-bucket
      patterns:
        - assets/*
      actions:
        - upload
        - delete
      prefix: assets
      acl: public-read
      metadata:
        foo: bar
        bar: foo
